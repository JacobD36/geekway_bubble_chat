🎯 Corrección de Alineación - Usuario a la Derecha

🔧 Problema identificado: Los mensajes del usuario se mostraban alineados a la izquierda en lugar de la derecha (como WhatsApp/Telegram)
❌ ANTES (Problema)
👤 Usuario: Izquierda
🤖 Bot: Izquierda
❌ Confuso y no estándar
✅ DESPUÉS (Corregido)
👤 Usuario: Derecha
🤖 Bot: Izquierda
✅ UX estándar de chat
➡️ Alineación Usuario
Mensajes del usuario ahora a la derecha
⬅️ Alineación Bot
Mensajes del bot mantienen posición izquierda
🔄 Flex Direction
row-reverse para usuario, row para bot
📱 UX Estándar
Sigue convención de apps de chat modernas
🎯 Justify Content
flex-end para usuario, flex-start para bot
📦 Margin Auto
margin-left: auto para empujar a derecha
